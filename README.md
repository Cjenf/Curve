# 📐Curve
### A  module using matplotlib for ploting Curves (Ellipse , Hyperbola , Parabola).
```py
pip install Curvify
```
## Example:
```py
import curve as cv
cv=cv.ellipse(3,3,5,5)
cv.draw()
print(cv.center) # output: (3,3)
```
